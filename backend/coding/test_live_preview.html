🚀 新功能特色

現在你可以透過 https://weakrisk.yinchen.tw/live-preview/{sessionId}/檔案路徑 直接存取工作區中的檔案，實現真正的即時預覽！

📁 支援的檔案類型

  • HTML 檔案 (.html, .htm)
  • CSS 樣式表 (.css)
  • JavaScript 檔案 (.js)
  • 圖片檔案 (.jpg, .jpeg, .png, .gif, .svg)
  • 字體檔案 (.woff, .woff2, .ttf)
  • 多媒體檔案 (.mp4, .webm, .mp3, .wav)

🔒 安全性保障

所有檔案存取都經過路徑安全性檢查，防止路徑遍歷攻擊，確保只能存取指定 session 目錄內的檔案。
